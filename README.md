# customer_shopping_behaviour_analysis
Data analysis project by using python, mysql, powerbi
📌 Project Overview

This project demonstrates end-to-end data analysis workflow using tools like Python, SQL, and Power BI.
It includes data cleaning, exploratory data analysis (EDA), SQL querying, and dashboard creation to extract meaningful insights.

🎯 Objectives
Clean and preprocess raw data
Perform Exploratory Data Analysis (EDA)
Write SQL queries to extract insights
Build interactive dashboards
Generate actionable business insights
🛠️ Tools & Technologies Used
🐍 Python
📊 Pandas
🗄️ MySQL
📈 Power BI
📓 Jupyter Notebook
